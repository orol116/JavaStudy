데이터 입출력

        요소의 내용을 입/출력

        [JS]
        - 요소.innerText          : 요소 내용 얻어오기 (태그 X)
        - 요소.innerText = "내용" : 요소 내용 출력하기 (태그 해석 X)

        - 요소.innerHTML          : 요소 내용 얻어오기 (태그 포함)
        - 요소.innerHTML = "내용" : 요소 내용 출력하기 (태그 해석 O)

        - input요소.value        : input 요소에 작성된 값 얻어오기
        - input요소.value = "값" : input 요소에 값 출력하기

        [jQuery]
        - $("단일 선택자").text()       : 요소 내용 얻어오기 (태그 X)
        - $("단일 선택자").text("내용") : 요소 내용 출력하기 (태그 해석 X)

        - $("단일 선택자").html()       : 요소 내용 얻어오기 (태그 포함)
        - $("단일 선택자").html("내용") : 요소 내용 출력하기 (태그 해석 O)

        - $("input요소 선택자").val()     : input 요소에 작성된 값 얻어오기
        - $("input요소 선택자").val("값") : input 요소에 값 출력하기

순회(탐색) 메서드

        [JS]
        - 요소.parentElement : 요소의 부모 요소 선택
        - 요소.children      : 요소의 모든 자식 요소 선택 (배열)

        - 요소.previousElementSibling : 요소의 이전 형제 요소 선택
        - 요소.nextElementSibling     : 요소의 다음 형제 요소 선택

        [jQuery]
        - $("단일 선택자").parent()              : 요소의 부모 요소 선택
        - $("단일 선택자").children()            : 요소의 모든 자식 요소 선택 (배열)
        - $("단일 선택자").children("CSS선택자") : 요소의 모든 자식 요소 중 CSS선택자가 일치하는 요소만 선택 (배열)

        - $("단일 선택자").prev() : 요소의 이전 형제 요소 선택 
        - $("단일 선택자").next() : 요소의 다음 형제 요소 선택


        parents()           : 모든 상위 요소
        parentUntil(선택자) : 현재 요소부터 선택자 요소까지의 모든 상위 요소
        find(선택자)        : 후손 중 선택자가 일치하는 모든 후손 요소
        siblings()          : 모든 형제 요소 선택

        prevAll()           : 이전 모든 형제 요소 
        nextAll()           : 다음 모든 형제 요소
        prevUntil(선택자)   : 이전 모든 형제 요소 중 선택자가 일치하는 요소
        nextUntil(선택자)   : 다음 모든 형제 요소 중 선택자가 일치하는 요소
